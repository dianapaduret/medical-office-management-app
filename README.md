# Medical Office Management App

A desktop application developed in C# using Windows Forms and .NET Framework for managing patients, doctors, diagnostics, and medical records.

## Features
- Patient management
- Doctor management
- Diagnostic management
- Search and filtering options
- Medical data organization through a graphical interface

## Technologies Used
- C#
- .NET Framework 4.7.2
- Windows Forms
- SQL Server Local Database

## Project Structure
- `Program.cs` – application entry point
- `Form1.cs` – login or main startup form
- `Home.cs` – home screen
- `Pacient.cs` – patient management
- `Doctor.cs` – doctor management
- `Diagnostic.cs` – diagnostic management

## How to Run
1. Clone the repository
2. Open `Proiect atestat 2024.sln` in Visual Studio 2022
3. Restore dependencies if needed
4. Make sure the local database is available
5. Run the project
